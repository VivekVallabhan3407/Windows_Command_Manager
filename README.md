# Windows Command Manager

A beginner-friendly tool to explore, understand, and practice Windows command-line commands.
Built without a database, using structured JSON files for simplicity and learning. Users can navigate to different sections using sidebar wiht easy navigation and good and simple design.


## Scope

- View categorized Windows commands

- Learn syntax, usage, and examples

- Practice basic command understanding wiht mcqs and scores.


## Features

- Command categories (File, Network, System, Disk, etc.)

- Search & filter commands by categories.

- Syntax + example viewer

- Practice mode for practicing random commands of all categories, wiht multiple type of formats.(non-scored)


## Tech Stack

- Frontend: React + JavaScript
- TailWindCSS: for styling
- React Toastify notifications
